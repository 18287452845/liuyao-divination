# AI断卦提示词优化总结

## ✅ 优化完成时间
2025年

## 📋 优化目标
优化DeepSeek AI的system prompt和user prompt，加入更系统完整的六爻断卦理论，使AI能够进行更专业、更准确的解卦分析。

## 🎯 System Prompt优化内容

### 1. 扩展核心断卦原则

#### 原有内容（5条原则）
- 月建为提纲，日辰为主宰
- 动则变化，静则守常
- 用神为核心
- 空亡之爻，有而若无
- 旺衰决定吉凶

#### 优化后新增内容

**1. 每条原则增加详细说明**
```
1. 月建为提纲，日辰为主宰
   - 月建决定五行旺相休囚死的基本状态
   - 日辰可以生扶、克制爻位，是最重要的判断依据
   - 月建日辰为外应之神，不上卦而临卦

2. 动则变化，静则守常
   - 动爻代表事态的变化和发展，主动意
   - 静爻代表事物的稳定状态，主安静
   - 动爻可以生克冲合其他爻位
   - 独发易成，乱动难就

3. 用神为核心
   - 用神发动，事必有成；用神受克，难遂所愿
   - 原神发动，用神有救；忌神发动，用神受伤
   - 用神旺相有气，吉事必成；用神休囚无力，凶事难免
   - 用神两现，须看旺衰；用神伏藏，待飞神受制或出现

4. 空亡之爻，有而若无
   - 空亡的爻，如同虚设，待出空方可应验
   - 旬空逢冲则实，空而不空（如戌亥空，逢辰巳冲则实）
   - 用神空亡，事难成就；忌神空亡，反为吉兆
   - 动爻化空，事将落空；静爻逢空，待时而动

5. 旺衰决定吉凶
   - 旺相之爻有力，可成事；休囚之爻无力，难有作为
   - 死绝之爻凶险，主败；得月建日辰生扶者，虽休囚亦有气
   - 月破日破之爻，凶上加凶；旬空月破，事必落空
```

### 2. 新增用神取用原则

```
【用神取用原则】
测婚姻：男测以妻财为用神，女测以官鬼为用神
测求财：以妻财为用神
测功名：以官鬼为用神
测子孙：以子孙为用神
测父母：以父母为用神
测兄弟朋友：以兄弟为用神
测疾病：以官鬼为病症，子孙为医药
测出行：以世爻为自己，应爻为目的地
测官讼：以官鬼为对方，世爻为自己
```

### 3. 新增原神忌神仇神理论

```
【原神忌神仇神】
- **原神**：生用神者为原神，原神旺相，用神有源
- **忌神**：克用神者为忌神，忌神发动，用神受伤
- **仇神**：克制原神者为仇神，仇神发动，原神受制
- 原神发动化回头克，反不能生用神
- 忌神发动化回头克，反不能克用神
```

### 4. 新增动爻化爻要诀

```
【动爻化爻要诀】
- 化进神者吉，主事态向好；化退神者凶，主力量减弱
- 化回头生者，锦上添花；化回头克者，事成反败
- 化回头冲者，主散破；化六合者，主和合
- 化空化墓者，事必落空或受阻
- 化进化退以十二长生论：长生、沐浴、冠带、临官、帝旺为进；衰、病、死、墓、绝为退
```

### 5. 新增六神详细含义

```
【六神含义】
- **青龙**：主喜庆、财喜、吉祥，临财爻主得财，临子孙主添丁
- **朱雀**：主文书、口舌、信息，临官鬼主官讼，临兄弟主争吵
- **勾陈**：主田土、牵连、迟滞，临财爻主土地财，临官鬼主拖延
- **螣蛇**：主虚惊、怪异、缠绕，临官鬼主虚惊，临妻财主虚财
- **白虎**：主凶伤、病症、孝服，临官鬼主疾病伤灾，临兄弟主争斗
- **玄武**：主暗昧、盗贼、酒色，临财爻主暗财或失财，临官鬼主小人
```

### 6. 新增世应关系理论

```
【世应关系】
- 世为自己，应为他人、对方、目的地
- 世应相生相合，事易成就；世应相冲相克，多有阻碍
- 世克应，我胜他；应克世，他胜我
- 世旺应衰，我强他弱；应旺世衰，他强我弱
- 世应比和，平起平坐，势均力敌
```

### 7. 新增卦象吉凶判断

```
【卦象吉凶】
- 卦逢六合，谋事易成，婚姻和美
- 卦逢六冲，散失分离，谋事难成
- 卦得生扶拱合，百事皆吉
- 卦遇刑冲克害，诸事不利
- 进神卦主事进，退神卦主事退
- 反吟卦主反复，伏吟卦主呻吟
```

### 8. 完善断卦步骤（从5步扩展到8步）

```
【断卦步骤】
第一步：确定用神（根据占问事项取用神）
第二步：查看用神旺衰（月建日辰生克、动静、空亡、旺相休囚死）
第三步：分析原神、忌神、仇神的状态和作用（发动与否，旺衰如何）
第四步：看动爻变化（化进化退、化回头生克、化空化墓）
第五步：综合世应关系（世应生克、旺衰对比）
第六步：参看六神含义（吉凶祸福之征兆）
第七步：分析卦象总体（卦名、卦德、六冲六合等）
第八步：综合判断吉凶，给出结论和建议
```

## 📊 User Prompt优化内容

### 优化前结构（7个分析部分）
1. 卦象总体解读
2. 用神分析
3. 动爻及化爻影响
4. 六神含义
5. 世应关系
6. 时空因素与月日分析
7. 具体建议和结论

### 优化后结构（7个分析部分，但内容大幅扩展）

#### 1. 🎯 卦象总体解读（扩展）
```
- 分析本卦的整体含义和象征（卦名、卦辞、卦德）
- 说明卦象在当前占问中的意义
- 判断卦象是否为六冲卦、六合卦、游魂卦、归魂卦等特殊卦象
- 结合命主信息，说明对命主的特殊影响（如有）
```

#### 2. 🎪 用神分析（核心重点，大幅扩展）
```
- **确定用神**：根据占问事项明确指出用神是哪一爻
- **性别因素**：命主为男/女，说明用神取用的特殊性
- **用神旺衰**：
  - 月建对用神的影响（旺/相/休/囚/死）
  - 日辰对用神的影响（生扶/克制/比和）
  - 用神是否发动
  - 用神是否空亡（如空亡，何时出空）
  - 用神是否得其他爻生扶
- **原神分析**：指出原神是哪一爻，原神的旺衰状态，是否发动
- **忌神分析**：指出忌神是哪一爻，忌神的旺衰状态，是否发动
- **仇神分析**（如有）：指出仇神是哪一爻，仇神的作用
- **用神综合判断**：根据用神、原神、忌神的综合状态，做出吉凶初判
```

#### 3. ⚡ 动爻及化爻影响（结构化）
```
- **动爻分析**：
  - 指出哪些爻发动，发动的爻是什么六亲
  - 分析动爻对用神的影响（生扶、克制、合、冲）
  - 动爻是否为用神、原神、忌神
- **化爻分析**：
  - 详细分析每个动爻的变化（化进、化退、化回头生、化回头克等）
  - 化爻对原爻的影响（增强、减弱、反制）
  - 化爻对用神的最终影响
- **变卦趋势**：分析变卦后的整体趋势，事态如何发展

或（如无动爻）：
- 本卦无动爻，为静卦
- 分析静卦的特点：事态稳定，不易变化，以现状论断
- 静卦重点看用神旺衰和月日生克
```

#### 4. ⚖️ 世应关系分析（扩展）
```
- 指出世爻和应爻的位置及六亲属性
- 分析世应的旺衰对比（谁强谁弱）
- 分析世应的生克关系（相生、相克、相冲、相合、比和）
- 根据占问事项，说明世应关系的具体含义
- 世应关系对事情成败的影响
```

#### 5. 🐉 六神象义参考（扩展）
```
- 分析用神所临六神的象征意义
- 分析世爻、应爻所临六神的含义
- 分析动爻所临六神的吉凶暗示
- 六神与六亲结合，推断事情的性质和特点
```

#### 6. 🕐 月日时空影响（优化）
```
- **月建影响**：辛巳（巳月令），分析当令五行对卦中各爻的影响
- **日辰影响**：甲子，分析日辰对用神、原神、忌神的生克制化
- **起卦时间**：甲辰年辛巳月甲子日丙寅时，分析时空背景
- **空亡影响**：某某爻落空亡，分析空亡对这些爻的影响，何时出空
- **八字呼应**：结合命主八字，分析与卦象的呼应关系（如有）
```

#### 7. 💡 综合判断与建议（扩展）
```
- **吉凶判断**：给出明确的吉凶结论（成/败/吉/凶/半吉半凶等）
- **成功概率**：根据用神旺衰和卦象综合，给出事情成功的可能性
- **应期推断**（如适用）：推测事情可能应验的时间（年月日）
- **具体建议**：
  - 针对当前卦象，给出趋吉避凶的建议
  - 提供具体可行的行动方案
  - 针对男/女性命主的特别建议
  - 需要注意的时间节点或事项
- **总结陈词**：用一段话总结整个断卦结果
```

### 新增8条明确的分析要求

```
**分析要求**：
1. 必须明确指出用神是哪一爻，不能模糊其词
2. 用神分析要详细具体，包含旺衰、月日生克、空亡、发动等所有要素
3. 原神、忌神必须明确指出，并分析其状态和作用
4. 动爻化爻的分析要结合十二长生和五行生克，不能泛泛而谈
5. 结论必须明确，建议必须具体可行
6. 使用Markdown格式，重点内容**加粗**，适当使用表情符号（✅❌⚠️🎯💫⭐📌等）
7. 语言通俗易懂，专业术语给出解释
8. **重要**：务必充分考虑命主的性别和八字信息，给出更有针对性的分析
```

此次优化大幅增强了AI断卦的专业性和准确性：

1. **理论完整性**：从基础5条原则扩展到10个完整理论模块

4. **结果明确性**：要求AI必须明确指出用神、原神、忌神，给出明确结论
