# 六十四卦方圆图说明

## 📊 表格布局

卦象速查功能已改为传统的8×8表格布局，遵循伏羲八卦次序。

### 布局规则

- **横向（列）**：外卦（上卦）
- **纵向（行）**：内卦（下卦）
- **单元格**：对应的卦象

### 八卦次序

表格按照以下八卦顺序排列：

| 序号 | 卦名 | 符号 | 性质 | 五行 |
|------|------|------|------|------|
| 1 | 乾 | ☰ | 天 | 金 |
| 2 | 兑 | ☱ | 泽 | 金 |
| 3 | 离 | ☲ | 火 | 火 |
| 4 | 震 | ☳ | 雷 | 木 |
| 5 | 巽 | ☴ | 风 | 木 |
| 6 | 坎 | ☵ | 水 | 水 |
| 7 | 艮 | ☶ | 山 | 土 |
| 8 | 坤 | ☷ | 地 | 土 |

## 🎯 使用方法

### 查找卦象

1. **定位上卦**：在表格顶部找到外卦（上卦）
2. **定位下卦**：在表格左侧找到内卦（下卦）
3. **查看卦象**：两者交叉处即为所求之卦

### 示例

要查找"水火既济"卦：
- 上卦：坎（☵ 水）→ 第6列
- 下卦：离（☲ 火）→ 第3行
- 结果：第3行第6列 = 水火既济（第63卦）

## 💡 传统意义

这种表格布局是传统易学中常用的64卦排列方式：

1. **伏羲八卦次序**：乾兑离震巽坎艮坤
2. **方位对应**：
   - 乾：西北
   - 兑：西
   - 离：南
   - 震：东
   - 巽：东南
   - 坎：北
   - 艮：东北
   - 坤：西南

3. **卦序规律**：
   - 对角线为八纯卦（乾为天、兑为泽、离为火等）
   - 横向/纵向对称位置为相错卦

## 🔧 技术实现

### 数据查询
```typescript
// 根据上下卦查找对应的卦
const findGua = (upperTrigram: string, lowerTrigram: string) => {
  return guaList.find(
    (gua) => gua.upperTrigram === upperTrigram &&
             gua.lowerTrigram === lowerTrigram
  );
};
```

### 表格渲染
- 外层循环：内卦（8行）
- 内层循环：外卦（8列）
- 动态查找并显示对应卦象
