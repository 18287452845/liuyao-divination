# 纳甲装卦法则参考

## 根据《京氏易传》和《卜筮正宗》

---

## 一、纳甲基本原则

### 阳卦阳支顺行，阴卦阴支逆行

**阳卦**：乾、震、坎、艮（卦象中阳爻多于阴爻）
**阴卦**：坤、巽、离、兑（卦象中阴爻多于阳爻）

## 三、重要说明

### 关于"乾纳甲壬、坤纳乙癸"

这是指配对关系：
- **乾纳甲壬**：乾为阳卦纳甲，其对宫坤为阴卦纳乙，甲与壬为阳干配对
- **坤纳乙癸**：坤为阴卦纳乙，其对宫乾为阳卦纳甲，乙与癸为阴干配对

**但在实际装卦时**：
- 乾卦六爻全部纳"甲"，不纳"壬"
- 坤卦六爻全部纳"乙"，不纳"癸"

"壬"和"癸"是在特殊情况下使用的，比如：
- 六壬课（另一种占卜体系）
- 某些派别的特殊装法

**传统主流六爻装卦法（《增删卜易》《卜筮正宗》）统一使用一个天干。**

## 四、装卦实例

### 例1：乾为天（乾宫本宫卦）

```
上爻 ━━━ 甲戌土 父母 （世）
五爻 ━━━ 甲申金 兄弟
四爻 ━━━ 甲午火 官鬼
三爻 ━━━ 甲辰土 父母 （应）
二爻 ━━━ 甲寅木 妻财
初爻 ━━━ 甲子水 子孙
```

注意：全部天干都是"甲"

### 例2：坤为地（坤宫本宫卦）

```
上爻 ━ ━ 乙酉金 子孙 （世）
五爻 ━ ━ 乙亥水 父母
四爻 ━ ━ 乙丑土 兄弟
三爻 ━ ━ 乙卯木 官鬼 （应）
二爻 ━ ━ 乙巳火 妻财
初爻 ━ ━ 乙未土 兄弟
```

注意：全部天干都是"乙"

### 例3：天风姤（乾宫一世卦）

上卦乾、下卦巽，但整体属于乾宫

```
上爻 ━━━ 甲戌土 父母
五爻 ━━━ 甲申金 兄弟
四爻 ━━━ 甲午火 官鬼
三爻 ━ ━ 辛酉金 兄弟
二爻 ━ ━ 辛亥水 子孙 （世）
初爻 ━ ━ 辛丑土 父母 （应）
```

注意：
- 上卦（乾）：仍然纳甲（甲戌、甲申、甲午）
- 下卦（巽）：纳辛（辛丑、辛亥、辛酉）
- 这是因为装卦时按照上下卦的卦象来装，不是按卦宫

## 五、关键结论

**装卦时的纳甲规则：**

1. 根据上卦和下卦分别装天干地支
2. 上卦是什么卦，就按照该卦的纳甲规则装上三爻
3. 下卦是什么卦，就按照该卦的纳甲规则装下三爻
4. 每个卦的六爻都是同一个天干（乾-甲、坤-乙、震-庚、巽-辛、坎-戊、离-己、艮-丙、兑-丁）

**卦宫归属是另一回事**，需要按照八宫六十四卦次序来判断，这会影响六亲和世应的安法。
